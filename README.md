
Conceptos clave: Kotlin básico, Composable functions, Layouts (Text, Image), alineación básica.
Crea una aplicación que muestre una foto de una mascota (o un animal favorito).
Usa un Box para poner una imagen de fondo y, encima, un Column que contenga el nombre del animal en un tamaño de fuente grande y una breve descripción debajo.
